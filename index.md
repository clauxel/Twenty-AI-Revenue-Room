# Twenty AI Revenue Room

Twenty AI Revenue Room is a SaaS workspace for Twenty CRM.

This repository is a public documentation project for Twenty AI Revenue Room. Its structure follows the public documentation pattern used by [MiroFish](https://github.com/clauxel/MiroFish): a short front door, a clear reading order, practical guides, reference pages, and a public-safe boundary.
